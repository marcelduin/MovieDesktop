It's 2018, the internet is filled with beautiful 60fps 4k video material. Why are we stil looking at the boring static desktop background?

Our PCs are fast enough...

Presenting MovieDesktop-- a Windows Only application that allows you to choose any video to play as your desktop background.

How to use

To use MovieDesktop, you have to have VLC Player installed on your system (32 or 64 bit).

Step 1: Download the program

Get the .ZIP file of MovieDesktop that matches your VLC installation:

MovieDesktop.32bit.zip
(x86, 294KB)
MovieDesktop.64bit.zip
(x64, 294KB)

Step 2: Unpack and run

Unzip the file in a new directory, and there, doubleclick on MovieDesktop.exe.

The first time you open it, it will directly ask you to select a video file to open. Select a file, and it will automatically start playing!

Step 3: More control

In your System Tray area, a MovieDesktop icon will be placed. If you right click it, you can:

  • Open a:
    • New video
    • Directory: you can select a directory from where a random video will be chosen
    • URL: You can enter a direct mp4 or webm URL which will be played
  • Select which screen the video runs on (if you have multiple monitors)
  • Quit the application

Run from command prompt

MovieDesktop can also be run from the command prompt. The first argument is the file, directory or URL to play, the second (optional) argument is the number of your desktop to play it on.

For example, to open an imgur mp4 as background on your second screen: 

MovieDesktop.exe https://i.imgur.com/VMb5aPE.mp4 2
